# Security Policy

## Supported Versions

| Version | Supported          |

## Reporting a Vulnerability

Please create an issue [here](https://github.com/ragarwalll/mta-forge/issues/new?assignees=ragarwalll&labels=bug&projects=&template=issue.md&title=) to report a vulnerability.
